# 📈 US Stocks ORB Trading Agent SaaS

A premium, deploy-ready Multi-User SaaS Platform for executing US Stocks Opening Range Breakout (ORB) strategy with automated Alpaca bracket order management.

## Features
- **Firebase Authentication**: Secure client-side Google OAuth sign-in.
- **Secure Encrypted API Keys**: Fernet AES-256 key encryption at rest inside Firestore.
- **Dynamic Charting**: In-line TradingView chart rendering dynamically upon stock selection.
- **Adjustable Parameters**: Custom Stop-Loss (SL) & Take-Profit (TP) parameters per user.
- **Performance HUD**: Real-time asset scan matrix, live execution logs, and active orders ledger.
- **Cumulative History Analytics**: Win rates, total realized P&L, streaks, and CSV trade logs exporter.
- **Aesthetic Dark/Light Theme Switching**: Premium persistent theme styling with smooth GSAP animations and WebGL 3D parallax background.
